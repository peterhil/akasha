# Akasha installation on Ubuntu Linux

```
sudo apt-get install libsdl1.2-dev \
  libsdl-image1.2-dev \
  libsdl-mixer1.2-dev \
  libsdl-ttf2.0-dev
sudo apt-get install libportmidi-dev \
  libsndfile1-dev \
  portaudio19-dev \
  libsamplerate0-dev \
  libsmpeg-dev

cd akasha-git
pip install -r install/test-requires.pip
```
